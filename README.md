📊 Analisis Stok Mobil Berdasarkan Merk, Warna, Area
Deskripsi
Proyek ini berfokus pada pengolahan data stok mobil menggunakan Microsoft Excel. Data dianalisis berdasarkan merk, warna, dan area untuk melihat distribusi stok di berbagai lokasi.

Dataset
Data terdiri dari:
- Merk mobil (Toyota, Hyundai, Nissan)
- Warna (Hitam, Putih, Silver)
- Area (Jakarta, Bandung, Surabaya)
- Jumlah stok

Metode yang Digunakan
- Pengelompokan data berdasarkan kategori
- Analisis multi-kriteria sederhana di Excel
- Pengolahan data berbasis filter dan rekap stok

Hasil Analisis
- Stok mobil dapat dikelompokkan berdasarkan merk
- Distribusi warna mobil di setiap area dapat dianalisis
- Perbandingan jumlah stok antar wilayah dapat diketahui

Tujuan Proyek
Proyek ini dibuat untuk:
- Melatih analisis data multi-kriteria
- Memahami pola distribusi data stok
- Mengembangkan kemampuan pengolahan data di Excel

Insight
Data dengan banyak variabel dapat memberikan informasi yang lebih mendalam. Dari analisis ini, terlihat bagaimana distribusi stok berbeda di setiap area dan berdasarkan karakteristik mobil.

Penutup
Proyek ini merupakan latihan analisis data lanjutan menggunakan Excel dan dapat dikembangkan lebih lanjut dengan visualisasi atau dashboard interaktif.
